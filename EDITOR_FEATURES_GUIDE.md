## Troubleshooting

### Common Issues

#### Images Not Uploading
**Symptoms:** Upload progress bar stuck or error message

**Solutions:**
1. Check file size (max 5MB)
2. Verify file type (JPEG/PNG/WebP/GIF only)
3. Ensure Supabase bucket exists and has RLS policies
4. Check network connection
5. Review browser console for errors

#### Paste Not Working
**Symptoms:** Cmd/Ctrl+V doesn't paste images

**Solutions:**
1. Ensure image is in clipboard (copy from image, not file)
2. Try using Upload button instead
3. Check browser clipboard permissions
4. Use drag-drop as alternative

#### Export Fails
**Symptoms:** Export button doesn't download file

**Solutions:**
1. Check browser download settings (allow downloads)
2. Disable popup blocker temporarily
3. Try different export format
4. Clear browser cache and retry

#### Color Picker Not Showing
**Symptoms:** Clicking "Pick Color" doesn't open picker

**Solutions:**
1. Ensure text is selected first
2. Check if menu is scrollable (scroll down)
3. Close emoji picker if open (only one can be open)
4. Refresh page if stuck

#### Page Breaks Not Exporting
**Symptoms:** Page breaks visible in editor but not in PDF

**Solutions:**
1. Ensure using latest version
2. Check browser print preview (Cmd/Ctrl+P)
3. Page breaks work best in PDF format
4. HTML export includes print CSS for page breaks

### Performance Tips

**For Large Documents (10+ pages):**
- Break into multiple smaller documents
- Optimize images before upload (compress externally)
- Avoid excessive inline images
- Use page breaks strategically

**For Slow Exports:**
- PDF export can take 5-10 seconds for long docs
- Use Markdown for fastest export
- Close other browser tabs
- Ensure stable internet connection

### Getting Help

**Resources:**
- Check browser console (F12) for error messages
- Review Supabase dashboard for storage issues
- Test in incognito mode (rules out extensions)
- Compare with fresh document (isolates doc issues)

**Bug Reports:**
Include:
- Browser and version
- Steps to reproduce
- Console error messages
- Screenshot if visual issue
- Document length and content type
